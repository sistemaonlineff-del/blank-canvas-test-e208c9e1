# Governança e Qualificação de Demandas - Bahia

Sistema em Streamlit com SQLite para cadastro de entidades, qualificação, cursos e fluxo administrativo.

## Rodar localmente

```bash
pip install -r requirements.txt
streamlit run app.py
```
